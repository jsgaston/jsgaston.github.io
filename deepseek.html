Instrucciones para integrar DeepSeek en tu página web

  1. Obtén tu clave API desde la plataforma de DeepSeek
  2. Guarda tu clave API en un archivo llamado deepseekapi.txt (solo la clave, sin espacios adicionales)
  3. Sube el archivo usando el formulario anterior o pega la clave manualmente
  4. Guarda la configuración para almacenar la clave en tu navegador (localStorage)
  5. Prueba la conexión enviando un mensaje a través del chat
  6. Para implementar en tu sitio, copia el código JavaScript de esta página y ajústalo según necesites
