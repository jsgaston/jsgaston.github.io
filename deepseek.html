Instrucciones para usar la API Key desde archivo

  1. Asegúrate de que el archivo deepseekapi.txt esté en tu servidor
  2. Haz clic en "Seleccionar archivo" para cargarlo desde tu dispositivo
  3. O haz clic en "Cargar desde Ruta del Servidor" para intentar cargarlo automáticamente
  4. Una vez cargada la API key, podrás probar la conexión con DeepSeek
  5. Si la conexión es exitosa, puedes enviar consultas a la API

Nota: Esta implementación simula la carga desde el servidor. En un entorno real, necesitarías implementar un endpoint en tu servidor que sirva el contenido del archivo.
