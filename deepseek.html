Solución de Problemas

Si no puedes conectarte, prueba lo siguiente:

  1. Verifica que tu API key sea válida y esté activa
  2. Prueba con diferentes endpoints en el selector
  3. Si usas un archivo TXT, asegúrate de que esté en el mismo directorio y solo contenga la clave API
  4. Revisa la consola del navegador (F12) para ver errores detallados
  5. Si persisten los problemas, contacta con el soporte de DeepSeek

Instrucciones de Uso

  1. Ingresa tu API key manualmente o haz clic en "Obtener API Key del Servidor"
  2. Selecciona un endpoint de la lista (prueba diferentes opciones)
  3. Selecciona el modelo que deseas utilizar
  4. Guarda la configuración
  5. ¡Comienza a chatear!
